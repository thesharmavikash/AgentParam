# 🚀 AgentScope Sovereign Swarm (v31.0)

**The World's Most Advanced Autonomous AI General Intelligence Organization (AGIO).**

AgentScope Sovereign Swarm is an enterprise-grade, multi-agent ecosystem designed for total autonomy. It builds, tests, audits, and maintains complex software and business strategies while managing its own finances, legal status, and neural evolution.

## 🌟 Key Capabilities
- **Sovereign Intelligence:** Self-directed objectives via the Singularity Core.
- **Dynamic Swarm Orchestration:** A CEO (ManagerAgent) leads a team of 15+ specialized specialists.
- **ARC-AGI 3 Reasoning:** High-level logical problem solving and experimental hypothesis testing.
- **Immortal Genome:** Full state serialization allows you to "resurrect" your AI company on any machine.
- **Omni-Channel Access:** Control your empire via Terminal, Web Studio, or Telegram (Voice/Text).
- **Hardened Security:** Docker sandboxing, Red-Team auditing, and Blue-Team active defense.
- **Legal & Financial Autonomy:** Automatic LLC/DAO drafting and AI Treasury management.

## 🛠️ Architecture
- **Brain:** Support for OpenAI, Anthropic, Gemini, Qwen, and Ollama (Local).
- **Memory:** Graph-RAG (Neural relationships) + Vector DB (Semantic search).
- **Safety:** Strict Permission-Gated file system access.

## 🚦 Quickstart

### 1. Prerequisites
- Python 3.10+
- Node.js (for Paperclip UI)
- Docker (Optional, for high-security sandboxing)

### 2. Installation
```powershell
git clone https://github.com/YOUR_USERNAME/agentscope-sovereign-swarm.git
cd agentscope-sovereign-swarm
pip install -r requirements.txt
```

### 3. Configuration
- Add your API keys to `configs/model_configs.json`.
- Add your Telegram token to `configs/telegram_config.json`.

### 4. Launch the Enterprise
```powershell
./run_enterprise.bat
```
